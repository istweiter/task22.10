Еще в 2016 году Microsoft анонсировала Unified Update Platform (UUP), чтобы сделать обновления Windows более плавными на всех устройствах. Компания утверждала, что UUP позволит сократить время проверки обновлений за счет уменьшения объема данных, отправляемых на клиентские устройства, а также сокращения времени обработки. Другими словами, больше обработки выполняется службой, а не на самом устройстве.

UUP была значительно улучшена с момента своего первоначального релиза, и теперь Microsoft готова выпустить публичную предварительную версию UUP в локальной среде.

Публичная предварительная версия UUP в локальной среде следует за приватной предварительной версией для зарегистрированных коммерческих клиентов в третьем квартале 2022 года. По сути, она действует как связующее звено между Windows Server Update Services (WSUS) и Microsoft Configuration Manager, обеспечивая быстрый и беспрепятственный процесс обновления.

UUP добавляет множество возможностей в процесс обновления Windows, включая обновление посредством одной перезагрузки, возможность установки необязательных функций и языковых пакетов, а также упрощенное управление содержимым с помощью методологий обслуживания.

Объявление о публичной предварительной версию UUP в локальной среде произошло в ключевой момент, поскольку ожидается, что платформа будет запущена в начале следующего года. После этой общей доступности все обновления для Windows будут доставляться через UUP.

Коммерческие клиенты, которые уже используют модель Software Updates , будут получать обновления UUP автоматически, но им рекомендуется удалить шаги обновления функций. Между тем, клиентам, которые все еще используют модель Task Sequence, использующую носитель ОС для выполнения обновлений функций, предлагается рассмотреть возможность перехода на модель Software Updates в ожидании появления UUP. Хотя организации по-прежнему смогут использовать Task Sequence, если захотят, им следует интегрировать ее с моделью Software Updates. Эта публичная предварительная версия — идеальное время для планирования тестирования этой миграции.
